# SecureShare üîê

SecureShare is a biometric-based secure file sharing application built using Flutter.

## Features
- Fingerprint authentication
- AES-256 encrypted file sharing
- SHA-256 integrity verification
- Owner-controlled download permissions
- File sharing history & traceability
- Secure local persistence

## Tech Stack
- Flutter & Dart
- local_auth
- AES Encryption
- SHA-256
- file_selector

## Platform
- Android
- Web (limited)
